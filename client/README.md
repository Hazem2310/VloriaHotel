# Veloria Hotel

Veloria Hotel is a full-stack web application for hotel management and reservations.

## Project Structure

- `client` – Frontend (React)
- `server` – Backend (Node.js + Express)

## Features

- Hotel rooms management
- Reservations system
- User authentication
- Admin dashboard (in progress)

## Technologies

- Frontend: React, CSS
- Backend: Node.js, Express
- Database: (to be added)
- Version Control: Git & GitHub
